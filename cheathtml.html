The Basics of Html:

"!Doctype Html": A way to declare your doc type Html.

"Html": Tells the browser the doc is written in Html.

"Head": Contains data about other data.

"Title": Displays name on the tab bar.

"Body": Contains everything seen on a webpage.

The Elements of Html:

"Link": Connects the Html doc to external sources.

"P": Creates Paragraph sections within the body.

"Div": Seperates content and data on webpage.

"img": Allows the addition of images.

The ways to use Html:

Every element must be contained by < and >.

The ending element needs a / after the first angeled bracket.

There are 6 different headings available.

An Id can be assigned to elements if it's used more than once and your using CSS.

A's are used as anchors so that two websites can be connected.

After an A the element href should be used to name the connected site.
