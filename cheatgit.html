The basics of every Git site:

To start in Git Hub you should create a repository.

You can click the code button to copy the link for the repository.

The command Git Clone "the link" allows you to also bring it to your desktop.

In Git Bash you can choose your directory by using the command cd and then spacing and typing whatever directory or file you pick.

A directory can be made by using the command mkdir and then spacing and typing the name you pick.

To place a repository inside a file Git Desktop can be utilized which has a direct feature for this.

Git Bash commands:

To view all files the command ls will allow you to do this.

Checking the status of a project can be achieved via the Git status command.

A file can be pushed by the command Git Push.

The help command allows for a list of possible commands to appear for anyone lost.

Git Reset can remove a file that has been saved.

The command rm means removes and is helpful to clear out your list of files.
